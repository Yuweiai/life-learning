## 内容提要

读者定位：

（1）刚刚进行网上炒股但实战经验较少的股民

（2）股票投资爱好者

（3）希望通过牛熊市操盘技巧提高自己稳步盈利能力的投资者

（4）作为大中专院校或者企业的网上炒股教材

（5）作为证券公司、基金公司等培训、指导客户和与客户沟通时的读本
